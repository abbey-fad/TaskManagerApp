# Task Manager App

## Overview

This is a simple **Task Manager** Android app built using **Kotlin** and **Jetpack Compose**. It allows users to **add tasks**, **view them in a scrollable list**, and **delete tasks** as needed.

**How to use the app:**
1. Open the app on your Android device.
2. Enter a task in the text field labeled “New task.”
3. Press the **Add Task** button to save it.
4. Scroll through the list of tasks.
5. Delete any task by pressing the **Delete** button next to it.

**Purpose:**  
The purpose of creating this app is to **practice Android development** with Jetpack Compose, **manage UI state in Kotlin**, and **build interactive mobile user interfaces**.
